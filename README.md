# WOLFz Clothing

Backend-first scaffold for a gamified 3D e-commerce platform.

## Quick Start

1. Copy env file:
   - `cp backend/.env.example backend/.env`
2. Start services:
   - `docker compose up --build`
3. Open API docs:
   - `http://localhost:8000/docs`

## Tech
- Backend: FastAPI, SQLAlchemy, PostgreSQL, Redis, Celery, JWT
- Frontend: React + Vite + Three.js (to be scaffolded next)
